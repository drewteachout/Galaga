Author: Drew Teachout

Game: Galaga

Goal: Shoot missiles to destroy enemies, dont't get hit by enemies or enemy missiles. When all enemies have been destroyed it proceeds to next stage (There is only one stage so it lets you play that stage again).

Controls:
	BUTTON_LEFT(Left Arrow): move ship left
	BUTTON_RIGHT(Right Arrow): move ship right
	BUTTON_A(Z): shoot a missile
	BUTTON_START(Enter): start game from start screen
	BUTTON_SELECT(Backspace): restart game from start screen;
